Design Notes
01 — SLASH COMMAND
Type /dailies to create
Artists type /dailies in any channel. A modal pops up for the session title + visibility. One submit → card drops into the channel.
02 — THE CARD
Visual anchor in the feed
Each daily gets a distinct card with a gradient stripe. CDs scrolling the channel can spot each artist's daily at a glance — no hunting through text.
03 — THUMBNAILS
See before you click
Up to 3 thumbnails inline, +N overflow. Locked/restricted assets show a 👥 icon instead of content. One click on any thumb deep-links into the Dailies viewer at that clip.
04 — VISIBILITY
Carried through from the app
The Team / Restricted / Private tag set in Dailies shows on the card. Restricted cards still appear in the channel — but locked assets are redacted.
05 — THREAD = NOTES
Timecoded replies in Slack
Notes left in Dailies appear as threaded replies beneath the card, with timecodes shown inline. The thread stays clean — CDs can skim notes without opening the app.
06 — UPLOAD UPDATES
Auto-posts as artists add files
When an artist uploads a new file to their daily throughout the day, a compact update is posted in the channel — keeping the team informed without creating noise.
07 — STATUS
Live on the card
Click the status in the card to cycle — New → In review → Approved. Updates instantly in both Slack and the Dailies app. No extra step.
